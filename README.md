# Stephen Jackson — Personal Site

This is my personal portfolio and resume site — a single place that brings together
who I am, the work I've done, and the things I've built.

It serves as a living resume: an overview of my professional background and
experience as a Senior Technical Program Manager, alongside a portfolio of personal
projects and experiments. It will continually be updated as my projects and
professional career expand, so it always reflects where I am and what I'm working on.

## What's here

- **About / Hero** — a short introduction and positioning statement.
- **Experience** — an editorial timeline of my professional career.
- **Portfolio** — small things I've built, playable right in the browser.
- **Contact** — how to get in touch.

## Built with

Plain HTML, CSS, and vanilla JavaScript, with [GSAP / ScrollTrigger](https://gsap.com/)
for scroll-driven motion.
